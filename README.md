# Product

A Product.slnLaunch file has been added so the API and UI run automatically in Visual Studio. If they do not start automatically, please follow the steps below:

Run from Visual Studio
1. Open the solution in Visual Studio 2026.
2. Right-click the solution → choose __Set Startup Projects...__.
3. Choose __Multiple startup projects__ and set the Action to `Start` for both `Product.API` and `Product.UI`.
4. Press F5 (or use __Debug > Start Debugging__).
